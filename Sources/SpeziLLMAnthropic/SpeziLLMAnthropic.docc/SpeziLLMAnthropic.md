Interact with Anthropic models in your app.

## Overview

See the [SpeziLLMOpenAI](https://swiftpackageindex.com/StanfordSpezi/SpeziLLM/main/documentation/SpeziLLMOpenAI) documentation for details.

- Important: SpeziLLM's Anthropic support is considered experimental, ans is implemented using the existing SpeziLLMOpenAI infrastructure, via [Anthropic's OpenAI compatibility layer](https://platform.claude.com/docs/en/api/openai-sdk).
    Not all features are necessarily supported; see Anthropic's documentation for more details.
