Interact with Gemini models in your app.

## Overview

See the [SpeziLLMOpenAI](https://swiftpackageindex.com/StanfordSpezi/SpeziLLM/main/documentation/SpeziLLMOpenAI) documentation for details.

- Important: SpeziLLM's Gemini support is considered experimental, ans is implemented using the existing SpeziLLMOpenAI infrastructure, via [Google's OpenAI compatibility layer](https://ai.google.dev/gemini-api/docs/openai).
    Not all features are necessarily supported; see Google's documentation for more details.
